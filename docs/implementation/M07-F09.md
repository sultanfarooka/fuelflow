## Key technical decisions

- **Preset:** `npx shadcn@latest apply --preset b3lo6Vpia` (R01).
  Replaces the hand-set orange token block (`--primary: 24.6 95% 53.1%`)
  with the preset's color, radius, spacing, and typography scales. The
  11 already-installed primitives are token-based and re-skin
  automatically ‚Äî no per-primitive edits required.
- **Primitive scope (R02):** install **Dialog, Sheet, Table, Tabs,
  Tooltip** ‚Äî completes the R02 list minus Form (see deviation). Already
  shipped: Button, Input, Select, Card, Badge, Alert, Label, Separator,
  Dropdown-Menu, Field, Sonner.
- **shadcn `Form` primitive ‚Äî INTENTIONALLY OMITTED.** Shadcn's `Form`
  wraps `react-hook-form` via `useFormContext`; the project uses
  TanStack Form via the existing `Field` + `FormTextField` stack. Adding
  shadcn `Form` would introduce a conflicting forms story and an unused
  `react-hook-form` dependency. The R02 row in the shipping PR is
  annotated with this deviation.
- **Success-state token:** add paired light/dark `--success` HSL tokens
  in `index.css` next to `--destructive`. Suggested starting values:
  light `--success: 142 70% 36%;` dark `--success: 142 65% 50%;` (tune
  to the preset palette during implementation). Document in
  `fuel-flow-web/src/components/CLAUDE.md`. The 4 green-using files
  migrate in Phase 5 to
  `bg-[hsl(var(--success)/0.1)] text-[hsl(var(--success))]`.
- **RTL utilities:** native Tailwind 3.4 logical utilities
  (`ms-`/`me-`/`ps-`/`pe-`/`start-`/`end-`/`text-start`/`text-end`/
  `border-s`/`border-e`/`rounded-s-*`/`rounded-e-*`). **No
  `tailwindcss-rtl` dependency.** `components.json` flip to
  `"rtl": true` makes future `shadcn add` runs emit logical utilities
  automatically.
- **`components.json`:** flip `"rtl": true`. Keep `style: "new-york"`,
  `baseColor: "neutral"`, `cssVariables: true`.
- **Dark-mode infra:** already correct ‚Äî `ThemeProvider` in `main.tsx`
  wraps the whole tree. The only gap is `<ModeToggle />` not being
  mounted on the auth layout; fixed in Phase 3.
- **RTL trigger:**
  `i18n.on('languageChanged', lng => document.documentElement.dir = lng === 'ur' ? 'rtl' : 'ltr')`
  wired in `main.tsx`. Initial `dir` set from persisted language on
  bootstrap so first paint is correct.
